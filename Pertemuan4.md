Deskripsi Program

Program ini merupakan implementasi pemodelan objek 3D sederhana menggunakan OpenGL. Objek yang dibuat adalah sebuah mobil 3D yang tersusun dari beberapa primitive object, seperti:

Kubus (Cube) sebagai badan mobil.
Kubus kecil sebagai atap mobil.
Bola (Sphere) atau Torus sebagai roda.
Bidang (Plane/Grid) sebagai lantai atau permukaan.

Program bertujuan untuk mempelajari konsep dasar:

Pemodelan objek 3D.
Penggunaan primitive object OpenGL.
Transformasi geometri.
Perspektif kamera.
Rendering wireframe dan solid object.
Tampilan Program

Program menampilkan sebuah mobil 3D berwarna ungu dengan:

Badan mobil.
Atap mobil.
Empat roda.
Lantai berbentuk grid sebagai acuan posisi objek.

Contoh tampilan:

Tampak samping mobil.
Tampak perspektif (3D view).
